## 📄 Summary
This course covers how to present and visualise data so that insights are easy to see and communicate. It works through the Python visualisation stack — Matplotlib, Seaborn, Folium, and Plotly — using a Canadian immigration dataset, before moving on to interactive dashboard creation with Plotly Dash. It concludes with a two-part [final assignment](Final%20Assignment/) analysing the impact of economic recessions on automobile sales, culminating in an interactive Dash dashboard.

|-------|-------------|

## 💡 Key Takeaway
> *A model or analysis is only as useful as someone's ability to understand it — the recession-impact dashboard shows how the same data can drive an interactive tool that lets a non-technical stakeholder explore trends themselves, rather than read a static report.*
