# Course 5 — Python Project for Data Science

## 📄 Summary
This course is a hands-on project course applying Python skills to a real-world data science scenario. Working as a Data Scientist / Data Analyst, the project involves extracting, cleaning, and visualising historical stock and revenue data for Tesla and GameStop — demonstrating the full Python data science workflow from data acquisition to dashboard creation.

## 📑 Main Topics
- Extracting stock data using the **yfinance** library
- Web scraping revenue data using **BeautifulSoup** and **Requests**
- Data cleaning: handling missing values, type conversion, and string manipulation
- Building interactive dashboards with **Plotly** and **Matplotlib**
- Comparing stock price trends against revenue performance
- Analysing the GameStop short squeeze as a case study in market anomalies

## 🔑 Key Skills Learned
- Fetching live financial data via APIs (yfinance)
- Web scraping tabular HTML data with BeautifulSoup
- Cleaning and transforming real-world financial datasets with Pandas
- Visualising time-series stock and revenue data
- Building multi-panel dashboards for financial storytelling
- Interpreting the relationship (and disconnection) between stock price and fundamentals

## 💡 My Takeaway
This project reinforced that data acquisition is rarely clean or straightforward. Both yfinance API changes and messy HTML tables required debugging and robust handling. The GameStop case study was particularly instructive — it demonstrates how social sentiment and retail investor coordination can completely decouple a stock price from its underlying business fundamentals, something no regression model would have predicted.
