Project Overview

Why Predicting Landings Matters

SpaceX has disrupted the space launch industry by recovering and reusing Falcon 9 first-stage boosters. Predicting whether a booster will land successfully has direct financial implications — it determines the true cost of a launch.

The Business Problem
SpaceX advertises Falcon 9 launches at $62M — far below competitors charging $165M+. The key: first-stage reusability. If competitors could predict when SpaceX will attempt a landing (and its likelihood of success), they could price launches more competitively.
Project Objective
Build a machine learning classifier that predicts whether the Falcon 9 first stage will successfully land, using publicly available launch data from the SpaceX API and Wikipedia. This enables competitive cost estimation for rival launch providers.
Dataset
90 Falcon 9 launches collected via the SpaceX REST API (v4) and Wikipedia web scraping. Features include flight number, payload mass, orbit type, launch site, booster version, and landing pad. Target: binary success/failure (Class 0/1).
Methodology
End-to-end data science pipeline: REST API data collection → web scraping → data wrangling → EDA → SQL analysis → interactive geospatial analytics with Folium → classification with GridSearchCV-tuned models.
⚠️ Key Insight
Each Falcon 9 booster reuse saves approximately $40–50M per flight. Accurate landing prediction directly informs competitive bidding, insurance underwriting, and SpaceX's own operational planning.

Data Visualization

Seven key visualizations reveal how flight number, payload mass, orbit type, launch site, and year correlate with landing success.

Flight Number vs. Payload Mass (Success/Failure)
Higher flight numbers (later launches) show an increase in successful landings. Payload mass alone does not determine success, but successful landings tend to cluster at medium payload ranges.
Flight Number vs. Launch Site
CCAFS LC-40 had the most early launches. KSC LC-39A shows a high success rate across all flight numbers. VAFB SLC-4E has no heavy payload launches (>10,000 kg).
Payload Mass vs. Launch Site
CCAFS LC-40 handles the widest payload range. VAFB SLC-4E is limited to lighter payloads. KSC LC-39A shows consistent success across payload masses.
Orbit Type Success Rate
ES-L1, GEO, HEO, and SSO orbits show 100% success rates. GTO has moderate success. SO has 0% success. Orbit type is a strong predictor of landing outcome.
Flight Number vs. Orbit Type
LEO orbit success is strongly correlated with flight number — more experience leads to better outcomes. GTO shows no clear trend with flight number.
Payload Mass vs. Orbit Type
Heavy payloads to Polar, LEO, and ISS orbits have higher success rates. GTO landings are less distinguishable by payload mass. VAFB handles lighter payloads exclusively.
Yearly Launch Success Rate Trend (2010–2020)
Success rate increased steadily from 2013, reaching near-perfect success by 2019–2020. Early years (2010–2012) had lower success rates as SpaceX refined techniques.

Python — Haversine Distance from math import sin, cos, sqrt, atan2, radians def calculate_distance(lat1, lon1, lat2, lon2): R = 6373.0 # Earth radius in km lat1, lon1, lat2, lon2 = map(radians, [lat1, lon1, lat2, lon2]) dlon, dlat = lon2 - lon1, lat2 - lat1 a = sin(dlat/2)**2 + cos(lat1) * cos(lat2) * sin(dlon/2)**2 c = 2 * atan2(sqrt(a), sqrt(1-a)) return R * c # KSC LC-39A coordinates launch_site_lat, launch_site_long = 28.56319, -80.57683 # Nearest coastline (same latitude, directly east) coastline_lat, coastline_long = 28.56319, -80.56794 distance_coastline = calculate_distance( launch_site_lat, launch_site_long, coastline_lat, coastline_long ) # → 0.87 km

Conclusions

Key Findings & Business Implications

Synthesizing insights across all five phases of the analysis.

What Predicts Success
  • Flight number is a strong predictor — success rate increases with launch experience
  • KSC LC-39A has the highest success rate among all launch sites
  • ES-L1, GEO, HEO, SSO orbits achieve 100% landing success
  • LEO landings improve significantly with flight number (experience effect)
  • Medium payload range (2,000–6,000 kg) correlates with higher success
  • Falcon 9 Block 5 (B5) is the most reliable booster version
Business Implications
  • ~$40–50M saved per reused booster — the model predicts this directly
  • Competitors can use predictions to underbid SpaceX on specific mission profiles
  • Insurance underwriters can price launch risk more accurately
  • SpaceX's success rate since 2019 approaches 100% — market-dominant position
  • VAFB SLC-4E unsuitable for heavy payloads — limits polar orbit capacity
  • Decision Tree model deployable as a real-time pricing API
Model Performance Summary
All four classifiers were trained on 80% of the 90-launch dataset (72 samples) and evaluated on the held-out 20% (18 samples). GridSearchCV with 10-fold cross-validation was used for hyperparameter tuning.
  • Decision Tree achieves 94.4% test accuracy — best overall performer
  • KNN achieves 88.9% — competitive second place
  • Logistic Regression and SVM both plateau at 83.3% — likely underfitting the non-linear decision boundary
  • The non-linear models (DT, KNN) outperform linear models, suggesting the launch success boundary is non-linear in feature space
  • The small dataset size (90 samples) limits confidence — larger datasets would further validate the Decision Tree advantage
